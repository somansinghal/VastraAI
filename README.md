# VastraAI 👗✨

AI-powered fashion ecommerce frontend built during my **15-Day Developer Challenge**.

Day **10 / 15**

VastraAI helps users discover sarees, manage carts, and checkout with a smooth modern UI.

https://vastra-ai-three.vercel.app
---

## 🚀 Features

### 🛍 Shopping

* 15 saree products
* Product detail modal
* Search functionality
* Wishlist system
* Cart system with quantity control

### ❤️ Wishlist

* Add to wishlist
* Remove from wishlist
* Move items to cart

### 🛒 Cart

* Increase / decrease quantity
* Remove item
* Empty cart
* Live cart total calculation

### 💳 Checkout

* Payment page
* Customer details form
* Order summary
* Order confirmation page

### 🧾 Invoice System

* Automatic bill generation
* PDF invoice download
* Order stored in localStorage

### 📦 Order Confirmation

* Shows:

  * Customer name
  * Address
  * Payment method
  * Order items
  * Total price

### 🎨 UI

* Mobile-responsive layout
* TailwindCSS styling
* Product cards
* Loading animation
* Clean ecommerce UI
